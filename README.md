# News to Comic

This repository contains a simple Python script that fetches the latest article from the New York Times RSS feed, produces a short summary, and generates a four-panel comic image.

## Requirements

Install dependencies using:

```bash
pip install -r requirements.txt
```

## Usage

Run the script with:

```bash
python news_to_comic.py
```

A `comic.png` file will be created in the current directory containing the summarized news in four panels.
